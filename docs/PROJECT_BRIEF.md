# Project Brief: AI Medical Assistant

## Application Area
**Healthcare & Medical Communication**
This project is an **AI-powered Medical Assistant** designed to bridge the gap between doctors and patients. It analyzes medical reports (images/PDFs) to provide:
- **For Doctors**: Detailed clinical summaries and entity extraction (diseases, meds).
- **For Patients**: Simplified, easy-to-understand explanations in their preferred language.

## Website & Technology
The application is a modern **Web App** with a responsive dashboard.

### Languages & Tools
- **Frontend (Website)**: 
  - **JavaScript** (React + Vite)
  - **CSS** (Tailwind CSS for styling)
- **Backend (Server)**: 
  - **Python** (Flask framework)
- **AI Services**: 
  - Google Cloud Vision (OCR)
  - AWS Comprehend Medical (Analysis)
  - Hugging Face (Summarization)
  - Azure AI Translator (Translation)
